105  The range of its value is :math:`[0, 255]`.
106 
107  Returns:
108  ~numpy.ndarray:
109  A preprocessed image.
110 
111  """
112  C, H, W = img.shape
113  scale1 = min_size / min(H, W)
114  scale2 = max_size / max(H, W)

116  img = img / 255.
